What is a QA Tester

A QA (quality assurance) tester helps make sure that clients or end users will not end up with faulty software. They do this by running tests to try to break the software or make it malfunction in as many ways possible, so that the developers will now what to fix before releasing their product.

Types of QA Testing
Unit testing QA testers begin with unit testing to evaluate individual software functions, allowing for easier identification of issues. Once unit testing is complete, they progress to component testing, which involves testing the interaction between multiple units of code.
Integration Testing QA testing ensures that different code components work together by testing them concurrently; if they don't function properly when integrated, developers must make adjustments.
